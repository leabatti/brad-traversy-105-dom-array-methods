# DOM Array Methods — Brad Traversy 50 Projects

Small project demonstrating the use of higher-order array methods in JavaScript.  
This project replicates the exercise from Brad Traversy’s *50 Projects in 50 Days* course.

## Features
- Add random users with a generated wealth value  
- Double all wealth values  
- Filter users by millionaire status  
- Sort users by wealth  
- Calculate total combined wealth  
- Dynamic DOM updates based on array operations  

## Technologies Used
- HTML5  
- CSS3  
- Vanilla JavaScript  

## Project Purpose
Practice JavaScript array methods such as `map`, `filter`, `sort`, `reduce`, and DOM manipulation.

## How to Use
1. Open `index.html` in your browser.  
2. Use the buttons to perform operations on the user list.
